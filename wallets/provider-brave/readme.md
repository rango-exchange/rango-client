# Brave
Brave Wallet integration for hub.  
[Homepage](https://brave.com/wallet/) | [Docs](https://brave.com/wallet/learn/)

More about implementation status can be found [here](../readme.md).

## Implementation notes/limitations

### Group
No group-related limitations have been identified in this integration.

### Feature

#### ⚠️ Cross Browser
Brave Wallet is **only available on the Brave browser**.  
It cannot be used on Chrome, Firefox, or other browsers.

All other features are fully supported with **no known limitations**.
